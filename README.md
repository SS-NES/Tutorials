# Tutorials

This repository is used to develop training materials for the following tools:

1. **meta-template** – A template system for research software projects.
   - Documentation: https://ss-nes.github.io/meta-template
   - Repository: https://github.com/SS-NES/meta-template

2. **code-auditor** – A tool for auditing research software code quality.
   - Repository: https://github.com/SS-NES/code-auditor

## Contribute

We welcome collaboration! Whether you want to improve existing tutorials, add new ones, fix typos, or suggest topics, your contributions are encouraged.

To get involved:
- Browse the [`tutorials/`](tutorials/) directory for existing and planned content.
- Open an issue to suggest a new tutorial or report a problem.
- Submit a pull request with your changes.

All skill levels are welcome. Let's build great training materials together!
